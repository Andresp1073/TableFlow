# TableFlow

**Professional Restaurant Reservation Management Platform**

TableFlow is a SaaS web application designed to streamline restaurant reservation management. The platform empowers restaurants to manage tables, reservations, customers, staff schedules, and business analytics through an intuitive and centralized dashboard.

---

## Features

- **Reservation Management** — Create, modify, cancel, and confirm reservations with real-time table availability.
- **Table Management** — Configure restaurant layouts, table capacities, and zones.
- **Customer Management** — Maintain detailed customer profiles, visit history, and preferences.
- **Staff Management** — Manage receptionists, waiters, and role-based permissions.
- **Multi-branch Support** — Operate multiple restaurant branches under a single account.
- **Real-time Availability** — Instant visibility of table status across all branches.
- **Notifications** — Email and in-app notifications for confirmations, reminders, and cancellations.
- **Reports & Analytics** — Dashboards with key metrics: occupancy rates, turn times, peak hours, and revenue.
- **Audit Logging** — Track every critical action across the platform.
- **Role-based Access Control** — Granular permissions for each user role.

| State Mgmt  | TanStack Query                              |

| Forms       | React Hook Form + Zod                       |
| Backend     | Node.js, Express, TypeScript                |
| ORM         | Prisma                                      |
| Database    | MySQL                                       |
| Auth        | JWT, Refresh Tokens, Bcrypt                 |
| API Docs    | Swagger                                     |
| Deployment  | Docker, Docker Compose, GitHub              |
