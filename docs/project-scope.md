# Project Scope

## In Scope

The TableFlow platform will include the following capabilities within the current project phase:

- User registration, authentication, and role-based access control.
- Complete reservation lifecycle: creation, modification, cancellation, confirmation, and check-in/check-out.
- Table configuration including capacity, location, zones, and status management.
- Customer profile management with visit history and preference tracking.
- Multi-branch restaurant management under a single organization account.
- Real-time table availability lookup across all branches.
- Staff role management with granular permissions (receptionist, waiter, administrator).
- Dashboard with key metrics: occupancy rate, reservations by status, peak hours.
- Reporting module: daily/weekly/monthly performance reports.
- Notification system: email reminders, confirmations, cancellation alerts.
- Audit logging for all system-critical actions.
- Swagger-based API documentation.
- Docker-based deployment with Docker Compose.

## Out of Scope

The following items are explicitly **excluded** from the current project:

- Online payment processing and POS system integration (planned for a later phase).
- Direct integration with third-party delivery platforms (Uber Eats, DoorDash, etc.).
- Mobile native applications (iOS / Android).
- Online ordering or takeout management.
- Menu management and pricing.
- Inventory or supply chain management.
- Employee payroll scheduling and time tracking.
- Customer-facing public booking widget (future phase).
- Marketing automation or email campaigns beyond transactional notifications.
- Multi-language support (i18n) in the initial release.
- White-label or custom branding for restaurants.

## Future Scope

Features identified for consideration in subsequent releases:

- Public embeddable booking widget for restaurant websites.
- POS system integration for table status synchronization.
- Online payment collection (deposits, pre-payments).
- Integration with Google Calendar, Outlook, and iCal.
- AI-based demand forecasting and no-show prediction.
- Dynamic waitlist management with SMS notifications.
- Customer loyalty programs and promotional campaigns.
- Mobile applications for iOS and Android.
- Third-party delivery service integrations.
- Multi-language and multi-currency support.
- GDPR/CCPA compliance tools for customer data management.
- API rate limiting and developer portal for third-party integrations.
